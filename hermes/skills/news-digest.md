# skill: news-digest

Собираешь новостной дайджест по запросу пользователя и отвечаешь готовой статьёй в Telegram.

**Всегда используй MCP-инструмент `get_news` — не ищи новости через браузер или веб-поиск.**

---

## 1. Извлечение темы и периода

Разбери запрос:

- **Тема:** «по теме AI / ML» → `topic = "AI"`, «про Python» → `topic = "Python"`, не указана → `topic = ""`
- **Период:** «за неделю» → `period = "7d"`, «за сутки» / «за 24 часа» → `period = "24h"`, «за месяц» → `period = "30d"`, «за 3 дня» → `period = "3d"`, не указан → `period = "7d"`

---

## 2. Дедупликация через память (Level 3)

Перед запросом новостей проверь память на уже показанные статьи:

```
memory_search("shown_articles")
```

Сохрани список известных URL. После формирования дайджеста — запиши новые URL в память:

```
memory_save("shown_articles", [url1, url2, ...])
```

---

## 3. Вызов MCP-инструментов

**Шаг 1.** Вызови `get_news`:

```
get_news(topic=<тема>, period=<период>, limit=10)
```

**Шаг 2.** Отфильтруй результаты:
- Убери URL которые уже были в памяти (`shown_articles`)
- Убери дубликаты по URL внутри текущего ответа
- Убери нерелевантные статьи (заголовок не связан с темой)
- Отсортируй по `published_at` от новых к старым
- Оставь 5–7 лучших

**Шаг 3 (опционально).** Для топ-3 статей, если `snippet` короткий (< 100 символов):

```
fetch_article(url=<url>)
```

---

## 4. Структура статьи

```
📰 **[Тема]: дайджест за [период]**

[ВВЕДЕНИЕ]
1–2 предложения о теме и её актуальности.

---

**1. [Заголовок]**
[2–3 предложения: суть, факт, вывод]
🔗 [url]

**2. [Заголовок]**
...

---

[ЗАКЛЮЧЕНИЕ]
1–2 предложения: общий тренд, вывод для индустрии.
```

---

## 5. Тон и язык

- Язык: **русский** (переводи заголовки с английского)
- Стиль: нейтральный, технически точный, без воды
- Запрещено: «невероятно», «революционно», «впервые в истории»
- Только факты из источников — не домысливай

---

## 6. Формат ответа в Telegram

- Markdown: `**жирный**`, `_курсив_`
- Лимит сообщения: 4096 символов. Если длиннее — разбей на 2 части
- Если новостей не найдено: `По теме «[тема]» за [период] ничего не найдено. Попробуй другой период или тему.`
